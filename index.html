    1Abstract Operations

    1.1Operations on Iterator Objects

    1.1.1GetIteratorDirect ( obj )

    1. If Type(obj) is not Object, throw a TypeError exception.
    2. Let nextMethod be ? GetV(obj, "next").
    3. Let iteratorRecord be Record { [[Iterator]]: obj, [[NextMethod]]: nextMethod, [[Done]]: false }.
    4. Return iteratorRecord.

    2Control Abstraction Objects

    2.1Iteration

    2.1.1The %IteratorPrototype% Object

    2.1.1.1%IteratorPrototype%.map ( mapper )

    1. Let iterated be ? GetIteratorDirect(this value).
    2. If IsCallable(mapper) is false, throw a TypeError exception.
    3. Let iterator be ObjectCreate(%IteratorPrototypeMapIteratorPrototype%, « [[Mapper]], [[Iterated]] »).
    4. Set iterator.[[Mapper]] to mapper.
    5. Set iterator.[[Iterated]] to iterated.
    6. Return iterator.

    2.1.1.1.1%IteratorPrototypeMapIteratorPrototype%

    The %IteratorPrototypeMapIteratorPrototype% object:

    • has a [[Prototype]] internal slot whose intrinsic value is %IteratorPrototype%.
    • is an ordinary object.

    2.1.1.1.1.1%IteratorPrototypeMapIteratorPrototype%.next ( v )

    1. Let O be this value.
    2. If Type(O) is not Object, throw a TypeError exception.
    3. If O does not have all the internal slots of a %IteratorPrototypeMapIteratorPrototype% Instance, throw a TypeError exception.
    4. Let mapper be O.[[Mapper]].
    5. Let iterated be O.[[Iterated]].
    6. Let next be ? IteratorStep(iterated, v).
    7. If next is false, then
      1. Return CreateIterResultObject(undefined, true).
    8. Let value be ? IteratorValue(next).
    9. Let mapped be Call(mapper, undefined, « value »).
    10. If mapped is an abrupt completion, return ? IteratorClose(iterated, mapped).
    11. Return CreateIterResultObject(mapped.[[Value]], false).

    2.1.1.1.1.2%IteratorPrototypeMapIteratorPrototype%.return

    The initial value of %IteratorPrototypeMapIteratorPrototype%.throw is %IteratorPrototypeReturnPass%.

    This property has the attributes { [[Writable]]: true, [[Enumerable]]: false, [[Configurable]]: true }.

    2.1.1.1.1.3%IteratorPrototypeMapIteratorPrototype%.throw

    The initial value of %IteratorPrototypeMapIteratorPrototype%.throw is %IteratorPrototypeThrowPass%.

    This property has the attributes { [[Writable]]: true, [[Enumerable]]: false, [[Configurable]]: true }.

    2.1.1.2.1%IteratorPrototypeFilterIteratorPrototype%

    The %IteratorPrototypeFilterIteratorPrototype% object:

    • has a [[Prototype]] internal slot whose intrinsic value is %IteratorPrototype%.
    • is an ordinary object.

    2.1.1.2.1.1%IteratorPrototypeFilterIteratorPrototype%.next ( v )

    1. Let O be this value.
    2. If Type(O) is not Object, throw a TypeError exception.
    3. If O does not have all the internal slots of a %IteratorPrototypeFilterIteratorPrototype% Instance, throw a TypeError exception.
    4. Let filterer be O.[[Filterer]].
    5. Let iterated be O.[[Iterated]].
    6. Repeat,
      1. Let next be ? IteratorStep(iterated, v).
      2. If next is false, return CreateIterResultObject(undefined, true);
      3. Let value be ? IteratorValue(next).
      4. Let selected be Call(filterer, undefined, « value »).
      5. If selected is an abrupt completion, return ? IteratorClose(iterated, selected).
      6. If ToBoolean(selected.[[Value]]) is true, return CreateIterResultObject(value, false).

    2.1.1.4.1.1%IteratorPrototypeDropIteratorPrototype%.next ( v )

    1. Let O be this value.
    2. If Type(O) is not Object, throw a TypeError exception.
    3. If O does not have all the internal slots of a %IteratorPrototypeDropIteratorPrototype% Instance, throw a TypeError exception.
    4. Let remaining be O.[[Remaining]].
    5. Let iterated be O.[[Iterated]].
    6. If remaining is 0, return ? IteratorNext(iterated, v).
    7. Set O.[[Remaining]] to 0.
    8. Repeat, while remaining > 0,
      1. Perform ? IteratorNext(iterated, undefined).
      2. Set remaining to remaining - 1.
    9. Return ? IteratorNext(iterated, v).

    2.1.1.6%IteratorPrototype%.reduce ( reducer, initialValue )

    1. Let iterated be ? GetIteratorDirect(this value).
    2. If IsCallable(reducer) is false, throw a TypeError exception.
    3. Let accumulator be initialValue.
    4. Repeat,
      1. Let next be ? IteratorStep(iterated).
      2. If next is false, return accumulator.
      3. Let value be ? IteratorValue(next).
      4. Let result be Call(reducer, undefined, « accumulator, value »).
      5. If result is an abrupt completion, return ? IteratorClose(iterated, result).
      6. Set accumulator to result.[[Value]].

    2.1.1.7%IteratorPrototype%.collect ( )

    1. Let iterated be ? GetIteratorDirect(this value).
    2. Let items be a new empty List.
    3. Repeat,
      1. Let next be ? IteratorStep(iterated).
      2. If next is false, return ? CreateArrayFromList(items).
      3. Let value be ? IteratorValue(next).
      4. Append value to items.

    2.1.1.8%IteratorPrototype%.forEach ( fn )

    1. Let iterated be ? GetIteratorDirect(this value).
    2. Repeat,
      1. Let next be ? IteratorStep(iterated).
      2. If next is false, return undefined.
      3. Let value be ? IteratorValue(next).
      4. Perform ? Call(fn, undefined, « value »).

    2.1.1.9%IteratorPrototype%.some ( fn )

    1. Let iterated be ? GetIteratorDirect(this value).
    2. Repeat,
      1. Let next be ? IteratorStep(iterated).
      2. If next is false, return false.
      3. Let value be ? IteratorValue(next).
      4. Let result be ToBoolean(? Call(fn, undefined, « value »)).
      5. If result is true, return true.

    2.1.1.10%IteratorPrototype%.every ( fn )

    1. Let iterated be ? GetIteratorDirect(this value).
    2. Repeat,
      1. Let next be ? IteratorStep(iterated).
      2. If next is false, return true.
      3. Let value be ? IteratorValue(next).
      4. Let result be ToBoolean(? Call(fn, undefined, « value »)).
      5. If result is false, return false.

    2.1.1.11%IteratorPrototype%.find ( fn )

    1. Let iterated be ? GetIteratorDirect(this value).
    2. Repeat,
      1. Let next be ? IteratorStep(iterated).
      2. If next is false, return undefined.
      3. Let value be ? IteratorValue(next).
      4. Let result be ToBoolean(? Call(fn, undefined, « value »)).
      5. If result is true, return value.

    2.1.2The %AsyncIteratorPrototype% Object

    2.1.2.1%AsyncIteratorPrototype%.map ( mapper )

    1. Let iterated be ? GetIteratorDirect(this value).
    2. If IsCallable(mapper) is false, throw a TypeError exception.
    3. Let iterator be ObjectCreate(%AsyncIteratorPrototypeMapIteratorPrototype%, « [[Mapper]], [[Iterated]] »).
    4. Set iterator.[[Mapper]] to mapper.
    5. Set iterator.[[Iterated]] to iterated.
    6. Return iterator.

    2.1.2.1.1%AsyncIteratorPrototypeMapIteratorPrototype%

    The %AsyncIteratorPrototypeMapIteratorPrototype% object:

    • has a [[Prototype]] internal slot whose intrinsic value is %AsyncIteratorPrototype%.
    • is an ordinary object.

    2.1.2.1.1.1%AsyncIteratorPrototypeMapIteratorPrototype%.next ( v )

    1. Let O be this value.
    2. Let promiseCapability be ! NewPromiseCapability(%Promise%).
    3. If Type(O) is not Object or O does not have all the internal slots of a %AsyncIteratorPrototypeMapIteratorPrototype% Instance,
      1. Let invalidIteratorError be a newly created TypeError object.
      2. Perform ! Call(promiseCapability.[[Reject]], undefined, « invalidIteratorError »).
      3. Return promiseCapability.[[Promise]].
    4. Let mapper be O.[[Mapper]].
    5. Let iterated be O.[[Iterated]].
    6. Let next be IteratorNext(iterated, v).
    7. IfAbruptRejectPromise(next, promiseCapability).
    8. Let done be IteratorComplete(next).
    9. IfAbruptRejectPromise(done, promiseCapability).
    10. If done is true, then
      1. Let iterResultObject be CreateIterResultObject(undefined, true).
      2. Perform ! Call(promiseCapability.[[Resolve]], undefined, « iterResultObject »).
      3. Return promiseCapability.[[Promise]].
    11. Let value be IteratorValue(next).
    12. IfAbruptRejectPromise(value, promiseCapability).
    13. Let mapped be Call(mapper, undefined, « value »).
    14. IfAbruptRejectPromise(mapped, promiseCapability).
    15. Set mapped to ? Await(mapped).
    16. Let iterResultObject be CreateIterResultObject(mapped, false).
    17. Perform ! Call(promiseCapability.[[Resolve]], undefined, « iterResultObject »).
    18. Return promiseCapability.[[Promise]].

    2.1.3%IteratorPrototypeReturnPass% ( value )

    The %IteratorPrototypeReturnPass% intrinsic is a built-in function object that is defined once for each realm.

    1. Let O be this value.
    2. If Type(O) is not Object, throw a TypeError exception.
    3. If O does not have a [[Iterated]] internal slot, throw a TypeError exception.
    4. Let iterated be O.[[Iterated]].
    5. Let return be ? GetMethod(iterated, "return").
    6. If return is undefined, return CreateIterResultObject(value, true).
    7. Return ? Call(return, iterated.[[Iterator]], « value »).

    2.1.4%IteratorPrototypeThrowPass% ( value )

    The %IteratorPrototypeThrowPass% intrinsic is a built-in function object that is defined once for each realm.

    1. Let O be this value.
    2. If Type(O) is not Object, throw a TypeError exception.
    3. If O does not have a [[Iterated]] internal slot, throw a TypeError exception.
    4. Let iterated be O.[[Iterated]].
    5. Let throw be ? GetMethod(iterated, "throw").
    6. If throw is undefined, return ThrowCompletion(value).
    7. Return ? Call(throw, iterated.[[Iterator]], « value »).
